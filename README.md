# 2D Heat Diffusion Simulation

This project simulates 2D heat diffusion using the finite difference method in Python.

## Features
- Solves the heat equation on a 2D grid
- Uses explicit time-stepping
- Uses NumPy for computation
- Uses Matplotlib for visualization

## Technologies
- Python
- NumPy
- Matplotlib

## Concepts
- Partial Differential Equations (PDEs)
- Finite Difference Method
- Time-stepping schemes
- Scientific Computing
